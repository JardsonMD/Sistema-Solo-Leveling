# Sistema Solo Leveling - Planejamento

## Objetivo
- Criar um sistema de tarefas gamificado inspirado em Solo Leveling
  - XP
  - Missões diárias
  - Evolução de níveis
  - Persistência de dados

## Funcionalidades atuais
- Menu interativo
- Missões diárias
- XP persistente
- Loop contínuo
- Reset diário automático

## Próximas funcionalidades
- Sistema de níveis baseado em XP
- Missões limitadas a uma vez por dia
- Recompensas e achievements
- Futuro: interface gráfica
- Automatizações no PC (PyAutoGUI)
- App mobile (integrar com Python ou outra linguagem)
- Streaks diárias e bônus por consistência
